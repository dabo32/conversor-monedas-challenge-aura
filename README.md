# Conversor de Monedas - JAVA 17

## Descripción

Estoy construyendo un Conversor de Monedas en Java que obtiene tasas de cambio en tiempo real mediante la API Exchange Rate API. Este proyecto me permitirá aprender a consumir datos desde una API, procesar respuestas en formato JSON y presentar la información de manera clara y útil para los usuarios.
Resolverá el problema de la conversión manual de monedas, ofreciendo tasas actualizadas de forma automática. Al integrar Exchange Rate API, aseguraré que los datos sean confiables y accesibles sin necesidad de cálculos externos
